Projects

Fetch | An instagram clone for dogs.

An full fledged instagram clone made specifically for posting pictures of dogs. It was built over a 4 week period created with a back end REST api using Node.js, Express, Sequelize, Postgres, PgAdmin 4, Bcryptjs, JsonWebToken, AWS, and S3. The front end was built with the ReactJs library. All component except for one are built with custom styling and styled to be responsive for both mobile and desktop. I took design inspiration from the Instagram app as well as a few sources online and some of my own ideas.

Doggo API | Vanilla JS Api fetching

This project was built with the dog api (https://thedogapi.com/). It works by connecting to the api by a vanilla javascript fetch and returning the data from the search in an array that can be paginated with the arrow buttons that appear on the sides of the card component when there is more than one result present. A picture of the breed also appear by a second fetch to a different api endpoint through string interpolation so that it will match the breed of the displayed text.

Netlfix | Static Layout

A very simple Netflix layout clone to demonstrate knowledge of positioning element on a webpage with css. This project was made over a period of a 4 days and was one of the first projects I had ever done involving web developement and design. It mainly uses css flexbox to position the images as well as some css animation to show the movie cards moving around on hover and transition.
